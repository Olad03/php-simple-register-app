# php-simple-register-app
### Setup Instructions
* Connect to your mysql db and create a new database, call it register.
* Edit the db.php file and provide your database credentials
* Connect to your database via a database client and import the db.sql file for the users table creation.
* Run the app, you can test locally by running the command `php -S localhost:9090`, the index.php will be served. 
* Visit localhost:9090 on a browser to interact with the app.
